# Brand logos

Drop official brand logo files here as `<slug>.png` (transparent background; a dark logo on
transparent reads best on the light tile). Then uncomment the matching line in
`lib/brandLogo.ts`. The brand row swaps the monogram for the logo automatically.

Slugs: lattafa, armaf, maison-alhambra, al-haramain, rasasi, afnan, swiss-arabian,
paris-corner, ard-al-zaafaran, french-avenue
